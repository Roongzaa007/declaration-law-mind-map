กลุ่มที่ 1 · เจตนาไม่ตรงกับใจ "ซ่อน-ลวง-พราง"

ผู้แสดงเจตนา รู้ตัว ว่าสิ่งที่แสดงออกไปไม่ตรงกับความตั้งใจจริง (มาตรา 154–155)

ประเภทคำอธิบาย / สูตรจำมาตรา 👆ผลทางกฎหมาย
เจตนาซ่อนเร้น
เก็บซ่อนในใจฝ่ายเดียว		 คิดอย่าง ทำอย่าง ฝ่ายเดียวทำเนียน
สูตร: "เก็บไว้ในใจ = สมบูรณ์ (เว้นแต่เขารู้ = โมฆะ)"		 154 👆 สมบูรณ์
เว้นแต่คู่กรณีรู้ → โมฆะ
เจตนาลวง
ฮั้วกัน 2 ฝ่าย หลอกชาวบ้าน		 สมรู้ร่วมคิดกันทั้งสองฝ่าย ทำนิติกรรมแต่ไม่ต้องการให้ผลนั้นเกิดจริง
สูตร: "ฮั้วกันหลอกชาวบ้าน = โมฆะ"		 155 ว.1 👆 โมฆะ
ยกสู้บุคคลภายนอกสุจริต+ค่าตอบแทนไม่ได้
นิติกรรมอำพราง
เปิดหน้า(ลวง) ซ่อนของจริงไว้		 ทำนิติกรรมบังหน้าเพื่อซ่อนนิติกรรมจริงไว้ข้างหลัง
สูตร: "เปิดหน้า=โมฆะ / หลังไมค์=บังคับตามนั้น"		 155 ว.2 👆 บังหน้า=โมฆะ
อำพราง=บังคับได้
📌 จุดระวัง ม.155 ว.1 — ถ้ามีบุคคลภายนอกที่สุจริตและเสียค่าตอบแทนเข้ามาเกี่ยวข้อง คู่กรณีจะยกความเป็นโมฆะขึ้นมาสู้กับบุคคลภายนอกคนนั้นไม่ได้

🔑 เทคนิคจำสำหรับข้อสอบ

ถามว่า "สัญญาที่แท้จริงคืออะไร?" → ดูม.155 ว.2 โยนส่วนที่เปิดหน้าทิ้ง แล้วนำส่วนที่ซ่อนอยู่ข้างหลังมาบังคับตามหลักกฎหมายที่เกี่ยวข้อง

🟠

กลุ่มที่ 2 · ความสำคัญผิด "ผิดสาระ vs ผิดสเปก"

แสดงเจตนาออกไปโดยไม่รู้ตัวว่าเข้าใจคลาดเคลื่อน (มาตรา 156–158)

ประเภทลักษณะ / ตัวอย่างมาตรา 👆ผลทางกฎหมาย
สำคัญผิดในสาระสำคัญ
"ผิดอย่างแรง = โมฆะ"		 ผิดระดับรากฐาน 3 กรณี:
① ผิดตัวบุคคล ② ผิดตัวทรัพย์ ③ ผิดลักษณะนิติกรรม
เช่น คิดว่าเซ็นเช่า แต่ดันเซ็นซื้อขาย		 156 👆 โมฆะ
เป็นโมฆะเด็ดขาด
สำคัญผิดในคุณสมบัติ
"ผิดสเปก = โมฆียะ"		 ตัวทรัพย์/คนถูก แต่คุณสมบัติไม่ตรงปก
เช่น ซื้อพระเครื่องคิดว่าแท้ที่จริงเก๊
หรือ จ้างคนคิดว่ามีใบอนุญาต แต่ไม่มี		 157 👆 โมฆียะ
บอกล้างได้
ประมาทเลินเล่ออย่างร้ายแรง
"สะเพร่าสุดๆ อดใช้สิทธิ์"		 มีสิทธิ์อ้างสำคัญผิด แต่ประมาทเลินเล่ออย่างร้ายแรงจนเกิดผิดนั้น
เช่น ไม่ยอมอ่านสัญญาเลย เซ็นแบบไม่ดู		 158 👆 อ้างไม่ได้
นิติกรรมสมบูรณ์
📌 จุดตัดสำคัญ ม.156 vs ม.157
ม.156 (โมฆะ) — ผิดใหญ่ ผิดตัวคน/ตัวทรัพย์/ชนิดนิติกรรม
ม.157 (โมฆียะ) — ผิดเล็ก คุณสมบัติไม่ตรง แต่รู้ว่าจะทำกับใคร ทำอะไร
💡 ผิดว่าพระแท้ที่จริงเก๊ = ม.157 โมฆียะ ไม่ใช่ ม.156
🔴

กลุ่มที่ 3 · ถูกแทรกแซง "กลฉ้อฉล & การข่มขู่"

แสดงเจตนาเพราะโดนปัจจัยภายนอกมากระทบ (มาตรา 159–165)

ประเภทองค์ประกอบ / สูตรจำมาตรา 👆ผลทางกฎหมาย
กลฉ้อฉล
"ถูกหลอกจนเปื่อย = โมฆียะ"		 ต้องเป็นกลฉ้อฉลถึงขนาด — ถ้ารู้ความจริงจะไม่ทำนิติกรรมนั้นเลย
ถูกหลอกถึงขนาด → โมฆียะ		 159 👆 โมฆียะ
บอกล้างได้ + เรียกค่าสินไหมได้
กลฉ้อฉลโดยบุคคลภายนอก
คนนอกเป็นคนหลอก		 บุคคลภายนอกหลอก → บอกล้างได้เฉพาะถ้าคู่กรณีรู้หรือควรรู้
คู่กรณีไม่รู้ → สมบูรณ์ แต่เรียกค่าสินไหมจากคนหลอกได้		 160 👆 โมฆียะ (เงื่อนไข)
กลฉ้อฉลเพื่อเหตุ
หลอกแค่บางส่วน ไม่ถึงขนาด		 ถูกหลอกแต่ถ้ารู้ความจริงก็ยังทำอยู่ดี เพียงแต่เงื่อนไขต่างออกไป
หลอกไม่ถึงขนาด → สมบูรณ์ แต่เรียกค่าสินไหมได้		 161 👆 สมบูรณ์
แต่เรียกค่าสินไหมได้
การข่มขู่
"กลัวภัยใกล้จะถึง = โมฆียะ"		 ภัยต้องใกล้จะถึงและร้ายแรงพอที่คนทั่วไปในสถานการณ์นั้นจะต้องกลัว
ขู่ภัยใกล้-ร้ายแรง → โมฆียะ		 164 👆 โมฆียะ
บอกล้างได้ภายใน 1 ปีนับแต่พ้นภัย
ข้อยกเว้น: ขู่ว่าจะใช้สิทธิ์ / เคารพยำเกรง
"ขู่แบบมีสิทธิ์ = ไม่ผิด"		 ① ขู่ว่าจะใช้สิทธิ์ตามกฎหมาย (เช่น ขู่จะฟ้องถ้าไม่ชำระหนี้)
② เคารพยำเกรงโดยธรรมชาติ (เช่น ยำเกรงบิดา)		 165 👆 สมบูรณ์
ไม่เป็นโมฆียะ
📌 จุดตัด กลฉ้อฉล vs ข่มขู่
กลฉ้อฉล (ม.159) = หลอกให้เชื่อสิ่งที่ไม่จริง → เหยื่อยินยอมแต่ถูกหลอก
ข่มขู่ (ม.164) = บังคับให้กลัวด้วยภัยร้ายแรง → เหยื่อยินยอมเพราะกลัว
💡 ทั้งคู่ผลเป็นโมฆียะเหมือนกัน แต่ที่มาและเงื่อนไขต่างกัน
🔵

กลุ่มที่ 4 · ผลของการแสดงเจตนา "ต่อหน้า vs ลับหลัง"

จังหวะเวลาที่กฎหมายถือว่าการแสดงเจตนา "เกิดผล" ผูกพันแล้ว (มาตรา 168–169)

ประเภทช่องทางมาตรา 👆จุดที่เกิดผล
แสดงเจตนาต่อหน้าบุคคล
"มีผลทันทีที่รับรู้"		 พูดโดยตรง, โทรศัพท์, วิดีโอคอล
ทุกช่องทางที่โต้ตอบกันได้ทันที		 168 👆 มีผลเมื่อรับรู้
เขาได้ยินและเข้าใจตอนไหน มีผลตอนนั้น
แสดงเจตนาลับหลังบุคคล
"มีผลเมื่อไปถึง"		 จดหมาย, อีเมล, SMS, เอกสาร
ทุกช่องทางที่ไม่โต้ตอบได้ทันที		 169 👆 มีผลเมื่อถึง
ถึงปุ๊บ มีผลปั๊บ ไม่ต้องรอให้เปิดอ่าน
การถอนการแสดงเจตนา
"ถอนก่อนถึง = ยังไม่มีผล"		 ส่งหนังสือถอนก่อนหรือพร้อมกันกับหนังสือแสดงเจตนา
เช่น ส่งอีเมลเสนอขาย แล้วโทรถอนก่อนอีกฝ่ายเปิดอ่าน → ถอนได้		 169 ว.2 👆 ถอนสำเร็จ=ไร้ผล
📌 ม.169 (ลับหลัง) — มีผลเมื่อไปถึง ไม่ใช่เมื่ออ่าน
ตัวอย่าง: ส่งจดหมายบอกเลิกสัญญา → จดหมายถึงบ้านปุ๊บ สัญญาเลิกปั๊บ แม้ยังไม่ได้เปิดอ่าน
⚠️ ถ้าผู้แสดงเจตนาตายก่อนจดหมายถึง — การแสดงเจตนายังคงมีผลตาม ม.169 วรรคท้าย

🔑 สูตรจำ ม.168 vs ม.169

ต่อหน้า (ม.168) = โต้ตอบได้ทันที → มีผลเมื่อรับรู้ (Hearing Rule)
ลับหลัง (ม.169) = ไม่โต้ตอบทันที → มีผลเมื่อถึง (Arrival Rule)
💡 ถามตัวเอง: "โต้ตอบกันได้ทันทีไหม?" ถ้าใช่ = ม.168 / ถ้าไม่ใช่ = ม.169
